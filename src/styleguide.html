Buttons

{% markdown %} Buttons are created by applying `.button` to an element. The button class is a utility class that applies padding, border, and background color. The button class can be combined with other utility classes to create different button styles. Changing button colors requires changing the background color, e.g. `bg-blue` to create a blue button. {% endmarkdown %}
